# Mohammad Haris — Portfolio

Personal portfolio website built as a single HTML file. Designed for AI/ML roles.

**Live at:** `https://m-haris7.github.io`

## Stack

Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build step.

## Features

- Terminal-style hero with JSON profile card
- Accordion experience section
- Scroll reveal animations + custom cursor
- Fully responsive (mobile-friendly)

## Setup

```bash
# Clone the repo
git clone https://github.com/M-Haris7/m-haris7.github.io

# Open locally
open index.html
```

No install or build required. Just open in a browser.

## Deployment

Hosted on **GitHub Pages**. Any push to `main` updates the live site automatically.
